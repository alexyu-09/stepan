# STEPAN Price List Converter

STEPAN — это мощный инструмент для обработки прайс-листов поставщиков (Excel/CSV) и их конвертации в стандартизированный формат для маркетплейсов.

![STEPAN Preview](https://via.placeholder.com/800x400?text=STEPAN+Interface+Preview)

## Основные возможности

- **Cloud Autopilot (NEW)**: Автоматическая проверка остатков в Google Sheets по расписанию (раз в сутки в 09:00 Kyiv) через серверный Cron.
- **Интеграция с Supabase**: Синхронизация ваших трекеров и настроек между устройствами в реальном облаке.
- **Интеллектуальный перевод (RU/UA)**: Полноценный машинный перевод (Google Translate) с кэшированием для быстрой обработки.
- **Маркетинговые приметки**: Автоматическое добавление условий опта и оплаты (безнал) во все товары.
- **Прямое редактирование**: Изменяйте данные прямо в таблице с предпросмотром.
- **Мониторинг наличия**: Отслеживание остатков в Google Sheets с уведомлениями в Telegram бот.
- **История сессий (IndexedDB)**: Автоматическое сохранение правок конвертера прямо в браузере.
- **Поиск изображений**: Встроенный поиск картинок в Google для каждого товара.

## Как запустить

### Вариант 1: Через Docker (Рекомендуется для скачивания Excel)

Этот способ гарантирует правильную работу скачивания файлов и стабильную работу веб-сервера.

1. Убедитесь, что у вас установлен [Docker Desktop](https://www.docker.com/products/docker-desktop/).
2. Выполните команду:
   ```bash
   docker-compose up -d --build
   ```
3. Откройте **[http://localhost:8080](http://localhost:8080)** в любом браузере.

### Вариант 3: Деплой на Vercel (Облако)

Самый быстрый способ запустить проект с работающим **Cloud Autopilot**:

1. Пушните код в свой GitHub.
2. Привяжите репозиторий к Vercel.
3. Добавьте переменные окружения: `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`, `SUPABASE_SERVICE_ROLE_KEY`.

## Технологии

- **Frontend**: React 19 + TypeScript + Vite.
- **Backend / Cron**: Vercel Serverless Functions.
- **База данных**: Supabase (PostgreSQL) + IndexedDB (Local).
- **Стили**: Чистый CSS (Modern glassmorphism).
